##Running  and visiting the website

Prerequisites:

Python (e.g. Python-2.7.12) installed


### Getting started

1. Run a local server

  ```bash
  $> cd /path/to/your-project-folder
  $> python -m SimpleHTTPServer 8080
  ```

2. Open a browser and visit localhost:8080
